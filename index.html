Kindergarten

Web application created for a course called Programsko Inžinjerstvo at University of Rijeka.
The app is designed to be used as a go to for kindergartens across the region. It enables parents to enroll children using an online interface, sign digital consents for children to participate in various events as well as follow their progress and grades.
Created using Django Framework.
Role: Project leader

FOODiY

Web application in form of a blog created for a course called Elektroničko Gospodarstvo at University of Rijeka.
In this app people can view or add their own recipes for making meals suitable for any period of the day. Users can create an account and participate in sharing, viewing, commenting, rating, adding to favorites and more.
Created using Django Framework.
Role: Lead programmer
